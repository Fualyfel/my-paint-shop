There are two ways to run this program

A. from Eclipse
Import -> Projects From Folders or Archives -> Archive -> MyPaintShop.zip -> Finish -> locate the package view and run the View class


B. from the jar
Open a CMD prompt and go to this folder.
run this command:
java -jar MyPaintShop.jar

Source code is contained in the .zip (refer to part A if you wish to see the code.)
